정규표현식 문자 매칭 방법

        [] : 대괄호 안의 어떤 문자와 일치
        /^[a,b,c,d]$/ a,b,c,d 중 하나라도 일치하는 문자가 있는지 검색
        [0-9] : 0부터 9까지 모든 숫자
        [ㄱ-힇] : ㄱ부터 힇까지 모든 한글
        [가-힇] : 가 부터 힇까지 (ㄱ-ㅎ ㅏ-ㅣ 는 제외)
        [a-z] : a-z 모든 영어 소문자
        [A-Z] : A-Z 모든 영어 대문자
        ------------------------------------------------

        [^] : 대괄호 안에 문자와 일치하지 않음
        [^abcd] : a,b,c,d 중에 하나라도 일치하는 문자가 없어야함 제외

        [A-Z]{5} : A~Z 사이에 5자리 입력
        [A-Z]{2,5} : A~Z 사이에 2~5자리 입력
        [A-Z]{2,} : A~Z 사이에 2자리 이상

        () : 그룹을 형성하고 일치하는 부분 문자열을 기억
        (19|20) : 19나 20과 일치하는가?

        \d : (0~9 중 아무 숫자)
        \s : (띄어쓰기, 엔터, 탭 등) 아무 공백 문자
